本项目选取了极少数在香港和美国上市的中国公司，组成自定义指数，尝试判断真实的中国经济市场指标与表现基准。其中香港上市公司组合为中国经济基本面指数，而美国上市公司组合为中国经济主要增量指数。

本项目使用老虎证券OpenApi，可参考[这篇文档](https://quant.itigerup.com/openapi/zh/python/overview/openWay.html)进行相关配置。
